# Groupon Operations Intelligence Command Center

An AI-powered operations intelligence system built for Groupon's Customer Operations team. The system ingests ~10,000 weekly support tickets, runs a multi-step analytical pipeline (data quality checks → trend detection → anomaly flagging → NLP theme extraction → opportunity sizing → recommendation generation), and produces a structured weekly "Ops Intelligence Brief" with actionable insights. The AI agent is built using the raw Anthropic Python SDK with tool use — no LangChain or agent frameworks.

## Key Findings

Analysis of 4 weeks of ticket data (9,776 clean tickets) identified **5 improvement opportunities worth ~$192K/yr**:

| # | Opportunity | Annual Savings |
|---|-------------|---------------|
| 1 | Expand chatbot deflection (27.9% → 43%) | $68K |
| 2 | AI agent co-pilot (reduce contacts 4.1 → 3.5) | $61K |
| 3 | Fix urgent ticket routing (244 misrouted to chatbot) | $35K |
| 4 | Phone-to-chat deflection (20%) | $20K |
| 5 | BPO Vendor B quality intervention | $8K |

**Critical anomaly:** 244 urgent tickets were misrouted to the AI chatbot, which costs $0.13/ticket but lacks the capability to handle urgent issues (vs. in-house at $3.60/ticket with significantly higher CSAT).

## Installation

```bash
pip install -r requirements.txt
```

## Environment Setup

```bash
cp .env.example .env
# Edit .env with your keys:
#   ANTHROPIC_API_KEY=your_key_here
#   SLACK_WEBHOOK_URL=your_webhook_here
```

## How to Run

Run each script in order:

```bash
# 1. Clean the raw data
python3 analysis/clean.py

# 2. Run exploratory analysis and opportunity sizing
python3 analysis/analyze.py

# 3. Export results to Google Sheets (requires credentials.json)
python3 analysis/sheets_export.py

# 4. Run the AI agent pipeline
python3 agent/pipeline.py

# 5. Launch the Streamlit dashboard
streamlit run app.py
```

## Tech Stack

- **Frontend:** Streamlit
- **AI Agent:** Anthropic Python SDK with tool use
- **Data Processing:** Pandas + NumPy
- **NLP:** Keyword/bigram extraction on customer messages
- **Notifications:** Slack Incoming Webhooks
- **Persistence:** JSON file for backlog state across weeks
